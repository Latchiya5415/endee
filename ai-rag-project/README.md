# AI RAG Project

## 📌 Description

This project demonstrates a simple Retrieval-Augmented Generation (RAG) system using Python.

## ⚙️ Features

- Reads data from a file
- Searches relevant content
- Displays ranked results

## 🛠️ Tech Stack

- Python

## ▶️ How to Run

```bash
cd ai-rag-project
python rag.py
```
